Starting a GPU for you
This demo runs on a real NVIDIA L4 that scales to zero when nobody is here, so an idle GPU costs nothing. Your visit just started one: the server boots, loads Qwen2.5-7B into GPU memory, then warms the kernels. Usually ready in about 2 minutes; a fully cold node can take 3.
